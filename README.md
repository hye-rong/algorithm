# algorithm

## 01 Greedy(그리디)
**그리디 알고리즘이란?**
- 현재 상황에서 지금 당장 좋은 것만 고르는 방법
- 문제에서 "가장 큰 순서대로" 과 같이 기준을 제시해줌
- 문제 풀이를 위한 아이디어를 떠올리고 정당성을 검토해야함
- ex) 거스름돈 문제에서 가장 큰 화폐부터 거슬러 주는 것
